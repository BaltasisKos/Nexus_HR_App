# 🌐 Nexus HR | Talent Synergy & Team Architect

**Nexus HR** is a high-performance Human Capital Management (HCM) engine designed to transform individual potential into collective excellence. It provides HR professionals and team leads with the data-driven insights needed to architect synchronized, high-performing teams.

---

## 🎯 The Vision
Nexus HR moves beyond traditional skill-tracking. It focuses on **Behavioral DNA**—identifying the innate patterns of thought and action that drive excellence. The platform enables organizations to bridge the gap between individual talent and organizational synergy, ensuring that every team is balanced, resilient, and strategically aligned.

## ✨ Key Features
* **Strengths Assessment Engine:** A randomized, bias-resistant assessment tool that identifies a user's "Top 5" dominant themes.
* **Talent Mapping:** Categorizes human capital into four strategic domains: *Executing, Influencing, Relationship Building, and Strategic Thinking*.
* **Synergy Analytics:** Visualizes how individual strengths complement one another to fill organizational gaps.
* **Fully Mobile Responsive:** Optimized for all devices, ensuring HR managers and employees can access talent insights on the go, from desktop to smartphone.

## 🛠️ Technical Stack
### Frontend
* **React.js (TypeScript):** For a robust, type-safe, and scalable user interface.
* **Axios:** Handling asynchronous API requests with interceptors for error management.
* **CSS3/Flexbox/Grid:** Custom-built responsive architecture (no heavy UI frameworks, ensuring maximum performance).
* **Mobile-First Design:** Implemented using CSS Media Queries and Fluid Grids to provide a seamless experience across all screen resolutions and aspect ratios.

### Backend
* **Python (Flask):** A lightweight yet powerful backend handling the scoring logic and randomized question distribution.
* **Flask-CORS:** Configured for secure cross-origin resource sharing between the development and production environments.
* **RESTful API:** Clean, structured endpoints for seamless data flow.
